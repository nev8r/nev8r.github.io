Operating System

一、计算机系统概述

1.1 操作系统的基本概念

1.1.1 操作系统的概念

  • 向上层提供方便易用的服务

    • GUI接口,调用程序接口实现
    • 联机命令接口 == 交互式命令接口 （用户讲一句，系统做一件句）
    • 脱机命令接口 == 批处理命令接口
    • 程序接口 ：在程序中进行 系统调用 来使用程序接口。普通用户不可以直接使用，只能通过程序间接使用。（系统调用 == 广义指令）
      • 系统调用的目的是请求系统服务

1.1.2 操作系统的四个特征

  • 并发性，共享性，虚拟性，异步性

  • 同时共享（并发访问）
  • 互斥共享

并发和共享的关系

  • 互为存在条件

  • 没有并发就没有共享

  • 没有共享就没有并发

  • 空分复用技术
  • 虚拟处理器技术（时分复用技术）（并发性）
  • 没有并发就没有虚拟性

  • 并发性才会导致异步性

1.3.3 系统调用

系统调用的过程

  1. 将系统调用号和所需要的参数压入堆栈
  2. 执行陷入指令，cpu用户态 -> 内核态 （执行中断）
  3. 分析系统调用类型，处理系统调用子程序
  4. 退出中断，恢复现场
