整除理论

概念

  • 定义：对于\(a,b\in{\mathbb{Z}}\)，如果\(\exists{b} \neq 0，b\in{\mathbb{Z}}\)，使\(a = b * q\)，则称\(b\)整除\(a\)，记为\(b \mid a\)，否则\(b \nmid a\)。

性质

  1. 若\(b \mid a\)，则有\(-b \mid a，b \mid -a，|b| \mid |a|\)。

    证明： \[ \because b \mid a\\ \therefore \exists b \neq 0，a = b * q\\ \therefore -a = b * -q\\ 则 b \mid a,其他同理 \] ​
